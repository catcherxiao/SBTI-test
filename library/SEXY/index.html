人格简介

SEXY（尤物）是 SBTI 性格测试里的标准人格之一。这个页面整理了它的一句话印象、完整设定、维度画像与相近类型，方便你在不重做测试的情况下直接了解这个人格的整体气质与典型表现。

您就是天生的尤物！ 这个一句话印象，通常最能快速对应到 SEXY（尤物）给人的第一观感。

完整设定

当您走进一个房间，照明系统会自动将您识别为天生的尤物，并自觉调暗亮度，以避免能源浪费。当您微笑时，您就变成了微笑着的尤物，周围的空气湿度也会显著下降，因为水蒸气都凝结成了人眼中的爱心。无论是谁，都容易对您的存在产生一种超标的注意力。传说，如果有足够多的SEXY人格聚集在一起开派对，其释放出的综合魅力能量足以暂时扭曲时空结构，让参加者产生“时间变慢了”的幸福错觉。他们不需要卖力表达，很多时候，单是存在本身就已经很像一篇华丽到过分的赋。

适合从哪类题看出来

SEXY 的标准维度画像是 HMH-HHL-HMM-HMM-HLH。从 15 维分布看，情感模型、社交模型 这几组题通常更容易把它和其他人格区分开，也更适合拿来观察你是否稳定靠近这个类型。

  • 情感模型：E1 依恋安全感偏高 / E2 情感投入度偏高 / E3 边界与依赖偏低
  • 社交模型：So1 社交主动性偏高 / So2 人际边界感偏低 / So3 表达与真实度偏高
  • 自我模型：S1 自尊自信偏高 / S2 自我清晰度中段 / S3 核心价值偏高
