人格简介

ATM-er（送钱者）是 SBTI 性格测试里的标准人格之一。这个页面整理了它的一句话印象、完整设定、维度画像与相近类型，方便你在不重做测试的情况下直接了解这个人格的整体气质与典型表现。

你以为我很有钱吗？ 这个一句话印象，通常最能快速对应到 ATM-er（送钱者）给人的第一观感。

完整设定

恭喜您，您竟然测出了这个世界上最稀有的人格。您或将成为金融界的未解之谜——是的，ATM-er不一定真的“送钱”，但可能永远在“支付”。支付时间、支付精力、支付耐心、支付一个本该安宁的夜晚。因此像一部老旧但坚固的ATM机，插进去的是别人的焦虑和麻烦，吐出来的是“没事，有我”的安心保证。您的人生就是一场盛大的、无人喝彩的单人付账秀。您竟用磐石般的可靠，承受了瀑布般的索取，偶尔夜深人静才会对着账单——可能是精神上的——发出一声叹息：我这该死的、无处安放的责任心啊。

适合从哪类题看出来

ATM-er 的标准维度画像是 HHH-HHM-HHH-HMH-MHL。从 15 维分布看，态度模型、自我模型 这几组题通常更容易把它和其他人格区分开，也更适合拿来观察你是否稳定靠近这个类型。

  • 态度模型：A1 世界观倾向偏高 / A2 规则与灵活度偏高 / A3 人生意义感偏高
  • 自我模型：S1 自尊自信偏高 / S2 自我清晰度偏高 / S3 核心价值偏高
  • 行动驱力：Ac1 动机导向偏高 / Ac2 决策风格中段 / Ac3 执行模式偏高
