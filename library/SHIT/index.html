人格简介

SHIT（愤世者）是 SBTI 性格测试里的标准人格之一。这个页面整理了它的一句话印象、完整设定、维度画像与相近类型，方便你在不重做测试的情况下直接了解这个人格的整体气质与典型表现。

这个世界，构石一坨。 这个一句话印象，通常最能快速对应到 SHIT（愤世者）给人的第一观感。

完整设定

恭喜您，SHIT人格是宇宙中已知的唯一一种稀有人格。所谓狗屎，并不是在抱怨，而是在进行一种神秘仪式。SHIT的行为模式是一场惊天动地的悖论戏剧。嘴上：这个项目简直是屎。手上：默默打开 Excel，开始建构函数模型和甘特图。嘴上：这帮同事都是 shit。手上：在同事搞砸之后，一边烦着，一边熬夜把烂摊子收拾得明明白白。嘴上：这个世界就是一坨 shit，赶紧毁灭吧。手上：第二天早上七点准时起床，挤上 shit 一样的地铁，去干那份 shit 一样的工作。别怕，那不是世界末日的警报，那是他马上要开始拯救世界的冲锋号。

适合从哪类题看出来

SHIT 的标准维度画像是 HHL-HLH-LMM-HHM-LHH。从 15 维分布看，情感模型、自我模型 这几组题通常更容易把它和其他人格区分开，也更适合拿来观察你是否稳定靠近这个类型。

  • 情感模型：E1 依恋安全感偏高 / E2 情感投入度偏低 / E3 边界与依赖偏高
  • 自我模型：S1 自尊自信偏高 / S2 自我清晰度偏高 / S3 核心价值偏低
  • 社交模型：So1 社交主动性偏低 / So2 人际边界感偏高 / So3 表达与真实度偏高
