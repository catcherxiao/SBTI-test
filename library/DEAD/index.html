人格简介

DEAD（死者）是 SBTI 性格测试里的标准人格之一。这个页面整理了它的一句话印象、完整设定、维度画像与相近类型，方便你在不重做测试的情况下直接了解这个人格的整体气质与典型表现。

我，还活着吗？ 这个一句话印象，通常最能快速对应到 DEAD（死者）给人的第一观感。

完整设定

恭喜您，您测出了全中国最为罕见的人格，只是“死者”这个名字实在有点晦气，所以也可以叫：Don't Expect Any Drives。死者已经看透了那些无意义的哲学思考，因此显得对一切“失去”了兴趣。死者们看世界的眼神，就像一位顶级玩家通关了所有主线、支线、隐藏任务，删档重开了999次之后，终于发现：这游戏压根就没意思。死者是超越了欲望和目标的终极贤者。他们的存在，就是对这个喧嚣世界最沉默也最彻底的抗议。

适合从哪类题看出来

DEAD 的标准维度画像是 LLL-LLM-LML-LLL-LHM。从 15 维分布看，行动驱力、自我模型 这几组题通常更容易把它和其他人格区分开，也更适合拿来观察你是否稳定靠近这个类型。

  • 行动驱力：Ac1 动机导向偏低 / Ac2 决策风格偏低 / Ac3 执行模式偏低
  • 自我模型：S1 自尊自信偏低 / S2 自我清晰度偏低 / S3 核心价值偏低
  • 态度模型：A1 世界观倾向偏低 / A2 规则与灵活度中段 / A3 人生意义感偏低
