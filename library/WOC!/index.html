人格简介

WOC!（握草人）是 SBTI 性格测试里的标准人格之一。这个页面整理了它的一句话印象、完整设定、维度画像与相近类型，方便你在不重做测试的情况下直接了解这个人格的整体气质与典型表现。

卧槽，我怎么是这个人格？ 这个一句话印象，通常最能快速对应到 WOC!（握草人）给人的第一观感。

完整设定

我们发现了一种神奇的生物——WOC!人。他们拥有两种完全独立的操作系统：一个叫“表面系统”，负责发出“我操”“牛逼”“啊？”等一系列大惊小怪的拟声词；另一个叫“后台系统”，负责冷静分析：嗯，果然不出我所料。WOC!人只会卧槽，不会多管闲事，因为他们深知，给傻逼讲道理，就像扶着烂泥上墙，不仅浪费体力，还弄自己一手屎。所以他们选择，握着一根智慧的大草，用一声饱含深情的“WOC！”来为这个疯狂的世界献上最高敬意。

适合从哪类题看出来

WOC! 的标准维度画像是 HHL-HMH-MMH-HHM-LHH。从 15 维分布看，自我模型、社交模型 这几组题通常更容易把它和其他人格区分开，也更适合拿来观察你是否稳定靠近这个类型。

  • 自我模型：S1 自尊自信偏高 / S2 自我清晰度偏高 / S3 核心价值偏低
  • 社交模型：So1 社交主动性偏低 / So2 人际边界感偏高 / So3 表达与真实度偏高
  • 行动驱力：Ac1 动机导向偏高 / Ac2 决策风格偏高 / Ac3 执行模式中段
