# 我的 SBTI 版本

这是一个基于原项目改出的个人版本，题库和测试逻辑保持原样，首页品牌信息改成了可自定义配置。

如果要继续改成更像你的版本，直接编辑 [index.html](/Users/catcherxiao/Desktop/codex/sbti/index.html) 里的 `SITE_CONFIG` 即可。

当前可配置项：
- 页面标题
- 首页主标题
- 当前版本标识
- 托管信息
- 域名信息

当前增强功能：
- 结果页支持一键保存最终测试结果长图
- 浏览器会自动缓存答题进度和最终结果，关闭后再次打开可继续查看
